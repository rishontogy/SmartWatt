SmartWatt is a smart web-based energy management system designed to help users monitor, analyze, and optimize electricity consumption. The project aims to promote sustainable energy usage by enabling users to track their electricity consumption, identify high-energy usage patterns, and adopt energy-saving practices.

```bash
git clone https://github.com/yourusername/SmartWatt.git
```

### 2. Open the Project

```bash
cd SmartWatt
```

### 3. Configure Database

- Create a MySQL database.
- Import the SQL database file.
- Update database credentials in the configuration file.

Example:

```php
$host = "localhost";
$dbname = "smartwatt";
$username = "root";

```

### 4. Run the Project

- Install XAMPP.
- Start Apache and MySQL.
- Copy the project folder into the `htdocs` directory.
- Open your browser and visit:
