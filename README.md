# Smart Home Energy Optimizer

## Project Description
The Smart Home Energy Optimizer is a web-based application designed to monitor and optimize energy consumption in a smart home environment. The system displays real-time device status and power usage, helping users reduce unnecessary energy consumption through data-driven insights.

## Technologies Used
- React.js (Frontend)
- Supabase (Backend Database and API)
- JavaScript
- HTML5
- CSS3
- GitHub Pages (Deployment)
